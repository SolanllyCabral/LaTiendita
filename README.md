# La Tiendita - Sistema de Inventario

## Descripción

"La Tiendita" es una aplicación de consola desarrollada en C# que permite gestionar un inventario de productos.  
El sistema permite registrar productos, actualizar stock, eliminar artículos y generar facturas de venta.

El proyecto está desarrollado de forma **procedural**, utilizando listas, estructuras de control (`if`, `while`, `switch`) y métodos.

---

## Funcionalidades

- Registrar productos
- Listar productos
- Actualizar stock
- Eliminar productos
- Generar factura de venta
- Validación de entradas
- Alerta de stock bajo (menos de 5 unidades)

---

## Requisitos

- .NET SDK instalado (versión 6 o superior recomendada)
- Visual Studio o Visual Studio Code

---

## Cómo ejecutar el programa

### Opción 1: Visual Studio

1. Abrir la carpeta del proyecto en Visual Studio
2. Buscar el archivo `.sln` o `.csproj`
3. Presionar **F5** o hacer clic en **"Iniciar"**
4. Se abrirá la consola con el menú del sistema

---

### Opción 2: Visual Studio Code

1. Abrir la carpeta del proyecto en VS Code
2. Abrir una terminal (`Ctrl + ñ` o `Ctrl + `)
3. Ejecutar el siguiente comando:
